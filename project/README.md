# MyBooks.in
#### Video Demo:  https://youtu.be/OQW0NyumeGU
#### Description:
The purpose of this project is to provide the user a platform to search for books and also bookmark the books to the user's account. It uses API from the openlibrary to fetch the data about the books for the app. When a user clicks Add to Bookmarks adds the book data to the database with the respective user_id and the books's data. The user also have the ability to remove the books from the database and to also change the user account password. It also uses another API to display a meme with a desired error message when the user encounters a error. The databse consists of two table namely user and bookmarks. The user table stores the username, user_id and hashed password of all the users. The second one stores the bookmarked books data with the respective user_id. The helper.py file contains all the essential functions among with are the ones which handle the API and return a list of dictionary containing the required data to the app.py and the the app.py file handles all the routes and logic handling.Thank You from Arman Ali Dafadar.
